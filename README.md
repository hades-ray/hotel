качаешь архивом, распаковываешь, закидываешь папку в C:\\xampp\htdocs\

открываешь эту папку в вс коде, открываешь файл hotel_db.sql, ctrl+a, копируешь весь текст что есть

запускаешь сервер в ксампе и переходишь в админку баз данных в браузере, базу не создавай самостоятельно, справа на главном экране панели увидишь вкладку SQL, жмешь появится окно в него втставляешь тот код что скопировал из файла sql, ниже в этом же окне будет кнопка вперед, должно все загореться зеленым - открываешь базу и смотришь таблицы

теперь в браузере пиши localhost/название_папки_проекта/index.php - главная страница, можешь зарегать для теста нового пользователя и тд
